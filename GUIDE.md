# Split AI Chat - 共有・デプロイガイド

オンライン接続成功おめでとうございます！
友人と一緒に使うための「共有方法」は大きく2つあります。

## 方法 A: Vercelに公開して共有（推奨）
遠くにいる友人と使うなら、この方法が一番です。

### 1. GitHubリポジトリの準備
ターミナルで以下のコマンドを順番に実行して、コードを保存します。
```bash
git init
git add .
git commit -m "Initial commit"
```
その後、[GitHub](https://github.com/new) で新しいリポジトリを作成し、表示されるコマンド（`git remote add...` から始まるもの）をターミナルで実行してプッシュします。

### 2. Vercelでデプロイ
1. [Vercel](https://vercel.com) にログイン。
2. "Add New..." -> "Project" をクリック。
3. GitHubのリポジトリを選択して "Import"。
4. **Environment Variables** (環境変数) の設定は**不要**です（コード内に直接書き込んだため）。
5. **"Deploy"** をクリック。
6. 完成したURLを友人に送れば、すぐに使えます！

---

## 方法 B: 同じWi-Fi内で共有（簡易）
友人が同じ部屋にいるなら、これだけでOKです。

1. あなたのPCの「ローカルIPアドレス」を調べます。
    *   **Mac**: ターミナルで `ipconfig getifaddr en0` を実行（例: `192.168.1.5` と出ます）。
2. 友人のPC/スマホのブラウザで、そのIPアドレスに `:3000` をつけてアクセスしてもらいます。
    *   例: `http://192.168.1.5:3000`
3. これでつながります！

## 補足: APIキーについて
現在、接続トラブル回避のためにAPIキーをコードに直接書き込んでいますが、本来はセキュリティ的によくありません。
テストが終わって本格的に公開する場合は、`lib/supabase.ts` の直接記述を元に戻し、Vercelの環境変数設定画面でキーを登録することを推奨します。
